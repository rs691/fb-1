# Woodify: Bespoke Wood Creations

Welcome to the Woodify demo application, a modern e-commerce and portfolio website designed for a custom woodworking business. This app showcases a complete user journey, from browsing products to making a purchase, all within a clean and responsive interface.

## Key Features

- **Dynamic Home Page**: A welcoming landing page featuring a prominent hero section, key feature highlights, and a curated list of featured products to immediately engage visitors.

- **Product Showcase**: A dedicated products page where customers can browse the full collection of handcrafted goods. Each item is presented in a card with its image, description, price, and an "Add to Cart" button.

- **Seamless Shopping Cart**: A fully functional shopping cart that allows users to add items, update quantities, or remove products. The cart is accessible from a slide-out sheet for quick viewing or on a dedicated cart page for a more detailed summary.

- **Integrated Checkout Flow**: A simulated checkout process powered by Stripe. Users are guided from their cart to a secure checkout form to complete their purchase.

- **User Authentication**: Secure user registration and login functionality using Firebase Authentication with email and password. The application's header intelligently updates to show the user's status.

- **Project & Event Pages**:
  - **Project Showcase**: A beautiful, masonry-style gallery to display past projects and highlight the quality of craftsmanship.
  - **Events Page**: A section to announce and detail upcoming workshops, craft fairs, or community events, complete with dates and locations.

- **Modern & Responsive UI**: Built with Next.js, React, and styled with Tailwind CSS and ShadCN UI components, the application is designed to look great on any device, from desktops to mobile phones. It also includes a theme-toggle for light and dark modes.
